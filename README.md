## Recipe website

Ссылка на проект: [https://recipes-next-beryl.vercel.app/recipes](https://recipes-next-beryl.vercel.app/recipes)

Сайт-справочник рецептов, реализованный на Next.js

На сайте реализованы следующие страницы:

1. [Страница авторизации](https://recipes-next-beryl.vercel.app/login)/[регистрации](https://recipes-next-beryl.vercel.app/register)
2. [Страница с карточками рецептов](https://recipes-next-beryl.vercel.app/recipes), фильтрами, пагинацией, поисковой строкой
3. [Страница карточки рецепта](https://recipes-next-beryl.vercel.app/recipes/24206ef2-55bc-4834-abb0-a9ebe3470254) с калькулятором ингредиентов (для расчета на определенное кол-во порций)
4. [Страница с созданием/редактированием собственных рецептов](https://recipes-next-beryl.vercel.app/my-recipes) (доступна только после авторизации)
5. [Страница с кратками описанием мотивации](https://recipes-next-beryl.vercel.app/about)

## Какие были использованны технологии:

* Next.js
* Next-auth
* React-hook-form
* React-toastify
* Tailwind
* Zod
* Prisma
